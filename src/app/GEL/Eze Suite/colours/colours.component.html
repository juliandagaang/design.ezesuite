• Colours

    All main style attributes including spacing, states and dimensions of Eze Suite's major components are defined in styles.scss as instructed in the introduction. However, all application-based colour properties are sorted out for easier usage and organisation. Hence, enable or import only one from the four application-based palettes according to the application that you are building. Base colours and status colours are global which are mainly utilised in styles.scss. 

  • Base colours

    Charcoal and mud are mainly used as foregrounds. Rock is mainly utilised to emphasize form inputs. Silver is applied as divider or border colours when required. Both ash and fog, and sometimes silver are utilised as container backgrounds.

  • Application-based colours

    There are four application-specific colour palettes which are blue for Eze Impress, orange for VMS, purple for QMS and green for Find. Each palette has three shades which are dark, main and light. Use dark colours for hyperlinks and light colours for backgrounds. Use the main colours (e.g. ez-blue) to define the overall look of an application.

    Enable style-blue.scss when using this palette.
